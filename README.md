マイ家計簿 Pro (Personal Finance Visualizer)
マイ家計簿 Pro は、日々の支出を素早く記録し、チャートで直感的に分析できるシングルページのWebアプリケーションです。
複雑な設定は一切不要。ブラウザを開くだけで、あなたの家計の「見える化」をサポートします。

🌟 主な特徴
直感的な支出分析: Chart.js を採用。支出カテゴリー（食費、交際費など）の割合をドーナツ型チャートでリアルタイムに表示します。

データ永続化: ブラウザの LocalStorage を利用しているため、ページを閉じたりリロードしたりしても、入力したデータが消えることはありません。

クイック入力: 日付（デフォルトで今日に設定）、カテゴリー、品目、金額をスムーズに入力できる最適化されたUI。

履歴管理機能: 支出履歴を日付順（降順）に自動ソートして表示。不要なデータはワンタップで削除可能です。

レスポンシブデザイン: スマートフォンでの操作を第一に考えた、モダンでクリーンなインターフェース。

🕹 使い方
支出の記録: フォームに「日付」「カテゴリー」「品目」「金額」を入力し、「追加する」ボタンを押します。

割合の確認: 画面上部のチャートで、どのカテゴリーに多く支出しているかを確認します。

履歴の管理: 画面下部の履歴リストから、過去の支出を振り返ることができます。削除したい場合は、右側の「×」ボタンを押してください。

🛠 技術スタック
Frontend: HTML5, CSS3, JavaScript (ES6+)

Library: Chart.js (CDN経由)

Storage: Browser LocalStorage API

🚀 セットアップ
サーバーの設置は不要です。

index.html をローカル環境に保存します。

Google Chrome や Microsoft Edge 等のブラウザでファイルを開きます。

Bash
# クローンして実行する場合
git clone https://github.com/your-username/my-budget-pro.git
cd my-budget-pro
open index.html
